Общая идея:
    1) приходит запрос на просмотр цены символов.
    2) Система смотрит есть ли по указанным символам данные в редис
    3) Если все есть - возвращает.
    Для недостоющих данных:
    4) Система смотрит есть ли открытый вебсокет на каком-либо из провайдеров (потом сделать несколько объектов/сокетов для каждего провайдера)
    5) Если нет, то открывается вебсокет, если есть - берется он
    6) По вебсокету делается запрос.
    7) Ответ записывается в redis и возвращается пользователю.
